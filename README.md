# Steven Hill Family & Faith Home

This is a starter static website for GitHub Pages.

## How to publish

1. Create a GitHub repository named `stevehill2683-hue.github.io`.
2. Upload all files from this folder.
3. In GitHub, go to Settings > Pages.
4. Under Build and deployment, select Deploy from a branch.
5. Choose the `main` branch and `/root` folder.
6. Your site should publish at `https://stevehill2683-hue.github.io`.
